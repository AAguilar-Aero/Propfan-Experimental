# Mission Requirements

**Status:** Working baseline

**Project Phase:** M1 - Requirements definition

**Last Updated:** July 2026

## 1. Purpose

This document establishes and defines the reference aircraft mission and initial requirements for the conceptual propfan propulsion system.

The purpose is not to design a certifiable aircraft or engine. These requirements provide a baseline for aircraft-performance, thermodynamic-cycle, gearbox and propfan analyses.

## 2. Reference Aircraft

The reference application is a twin-engine regional passanger aircraft designed for short and medium-range routes.

| Parameter | Working value |
| --- | ---: |
| Passenger capacity | 90 passengers |
| Aircraft configuration | Twin-engine regional aircraft |
| Propulsion architecture | Geared, variable-pitch, single-rotor open propfan |
| Number of propulsors | 2 |
| Design objective | Reduced mission fuel consumption while retaining regional-aircraft capability |

## 3. Mission Design

The mission design represents the typical regional route

| Parameter | Working value |
| --- | ---: |
| Design range | 1,500 NM|
| Cruise Mach Number | 0.72 |
| Cruise Altitude | 9000 m |
| Mission Segments | Taxi, Take-off, Climb, Cruise, Descent, Landing and reserves |
| Payload | 90 passengers with baggage (value to be defined) |
| Diversion and holding reserve | To be decided during mission analysis |
