# Python package

The package will be organised by engineering domain as modules are implemented:

- `atmosphere`: ISA and off-standard atmosphere calculations.
- `aircraft`: mission and aircraft-performance models.
- `cycle`: gas-turbine shaft-power cycle calculations.
- `gearbox`: transmission sizing and losses.
- `propeller`: geometry, BEMT and airfoil interfaces.
- `optimisation`: design-space exploration and Pareto analysis.
